## 安装




使用cnpm i -g yarn 或 npm i -g yarn安装yarn
配置registry地址：yarn config set registry http://registry.npm.taobao.org
配置sass-binary-site地址：yarn config set sass-binary-site http://npm.taobao.org/mirrors/node-sass

yarn install

npm start

##构建
npm run build
